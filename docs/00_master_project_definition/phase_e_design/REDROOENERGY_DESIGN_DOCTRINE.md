# RedRooEnergy Design Doctrine

Version: v1.0 DRAFT
Governance State: CORE + EXTENSIONS LOCKED
Enforcement State: LIVE

Purpose:
Define the authoritative design principles governing all visual,
layout, and merchandising decisions for the RedRooEnergy platform.

Reference Model (LOCKED):
- Marketplace structure & mechanics: Newegg (AU)
- Visual tone, softness, colour restraint: First Option Bank

Synthesis Rule:
Newegg’s commercial clarity without visual aggression.
First Option Bank’s calm confidence without retail softness.

Design Principles:
- Calm, professional, trustworthy
- Soft edges, rounded geometry
- Muted colour palette
- Clear hierarchy, generous spacing
- Confidence through clarity, not force

Forbidden:
- Harsh reds, sharp angles, visual noise
- Aggressive discount styling
- Gimmicks or trend UI patterns
- Dense, cluttered layouts

Accessibility:
- Readable typography
- Clear contrast
- Predictable interaction patterns

Authority Rule:
Design must express governance, not excitement.
