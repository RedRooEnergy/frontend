# Finance & Settlement Events — Contract Definition

This document defines Finance Authority–originated audit events.

Events represent authoritative financial decisions
or observational financial review actions.

Examples (non-exhaustive):
- FINANCIAL_CASE_VIEWED
- ESCROW_STATE_VIEWED
- SETTLEMENT_STATE_VIEWED
- ESCROW_RELEASE_AUTHORISED
- SETTLEMENT_FINALISED
- REFUND_AUTHORISED
- ADJUSTMENT_AUTHORISED
- DISPUTE_RESOLVED

All events are auditable and immutable.
