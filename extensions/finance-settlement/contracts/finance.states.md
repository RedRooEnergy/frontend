# Finance Authority–Visible States — Contract Definition

This document defines Finance Authority–visible projections
of Core financial and settlement state.

Examples:
- PAYMENT_RECEIVED
- ESCROW_HELD
- ESCROW_RELEASE_AUTHORISED
- SETTLEMENT_IN_PROGRESS
- SETTLEMENT_COMPLETED
- REFUND_AUTHORISED
- REFUND_COMPLETED
- DISPUTE_OPEN
- DISPUTE_RESOLVED

States are projections only and must reflect Core truth.
