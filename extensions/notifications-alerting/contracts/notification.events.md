This document defines notification lifecycle audit events.

Events cover creation, dispatch, delivery, suppression,
and user interaction with notifications.

Examples (non-exhaustive):
- NOTIFICATION_CREATED
- NOTIFICATION_TEMPLATE_RESOLVED
- NOTIFICATION_DISPATCH_ATTEMPTED
- NOTIFICATION_DELIVERED
- NOTIFICATION_DELIVERY_FAILED
- NOTIFICATION_VIEWED
- NOTIFICATION_SUPPRESSED

All events are auditable and immutable.
