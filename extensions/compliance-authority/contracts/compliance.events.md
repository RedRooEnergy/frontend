# Compliance Authority Events — Contract Definition

This document defines Compliance Authority–originated audit events.

Events are authoritative decisions or observational actions
and must always emit audit records.

Examples (non-exhaustive):
- COMPLIANCE_CASE_VIEWED
- COMPLIANCE_EVIDENCE_VIEWED
- COMPLIANCE_DECISION_ISSUED
- COMPLIANCE_DECISION_APPROVED
- COMPLIANCE_DECISION_REJECTED
- COMPLIANCE_DECISION_SUSPENDED
- COMPLIANCE_DECISION_REVOKED

All events are auditable and immutable.
