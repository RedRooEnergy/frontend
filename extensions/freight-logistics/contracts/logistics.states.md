# Logistics Operator–Visible States — Contract Definition

This document defines Freight & Logistics Operator–visible projections
of Core shipment and consignment state.

Examples:
- SHIPMENT_CREATED
- SHIPMENT_IN_TRANSIT
- SHIPMENT_AT_PORT
- SHIPMENT_DELIVERED
- CONSIGNMENT_LOADED
- CONSIGNMENT_UNLOADED
- CONSIGNMENT_DAMAGED

States are projections only and must reflect Core truth.
