# Logistics Operator Events — Contract Definition

This document defines Freight & Logistics Operator–originated audit events.

Events are observational or signalling only and must not
directly trigger authoritative state changes.

Examples (non-exhaustive):
- SHIPMENT_LIST_VIEWED
- SHIPMENT_VIEWED
- CONSIGNMENT_VIEWED
- SHIPMENT_STATUS_SIGNALLED
- CONSIGNMENT_STATUS_SIGNALLED
- SHIPMENT_EXCEPTION_REPORTED

All events are auditable and immutable.
