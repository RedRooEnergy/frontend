# Fulfilment and Shipment Governance
Version: v1.0 (LOCKED)  
Scope: Governance-only; no implementation

## Required Shipment Milestones
- Pickup confirmed (timestamp, location, carrier).
- Export clearance completed (where applicable).
- In-transit with tracking ID.
- Delivery confirmation (POD) with timestamp.

## Evidence Requirements
- Carrier tracking, POD, and any customs documents must be stored and auditable.
- Milestone evidence must be linked to order ID and timestamped.

## Buyer Visibility Rules
- Buyers must see milestone status and timestamps; no raw customs documents exposed unless required.
- Refund/return windows start on delivery confirmation timestamp.

## Payment Release Dependencies
- Settlement eligibility requires delivery confirmation and expiry of return window, unless admin override.
- If dispute/return/warranty active, settlement is paused until resolved.
