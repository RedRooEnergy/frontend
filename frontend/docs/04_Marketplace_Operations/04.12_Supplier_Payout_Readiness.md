# Supplier Payout Readiness
Version: v1.0 (LOCKED)  
Scope: Governance-only; no implementation

## Banking Verification
- Beneficiary matches KYB entity.
- Account validated via small transfer or provider verification (Wise/Airwallex).
- Currency and country supported by approved rails.

## Tax / Export Documentation
- Tax identifiers (ABN for AU suppliers; equivalent for overseas).
- Export/import licences where applicable.
- HS codes declared for shipped goods.

## Conditions Preventing Payout
- Active disputes/returns/warranties.
- Missing delivery proof or incomplete compliance docs.
- Sanctions/PEP match pending review.
- Refund in progress.

## Escalation Paths
- Tier 1: Ops review; request missing evidence.
- Tier 2: Compliance review; possible payout block.
- Tier 3: Admin override (logged with approval ID).
