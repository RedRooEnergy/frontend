## Required Audit Exports
- Payment events: initiation, webhook confirmations, refunds, settlements, overrides.
- Compliance events: approvals/rejections and linked evidence.
- Disputes/returns/warranties: state changes with timestamps.

## Payment Trace Formats
- Include order ID, payer/buyer, supplier, method, currency, amounts, FX rate (if any), timestamps, approval IDs.
- Export in CSV/JSON; integrity via hash of export file.
