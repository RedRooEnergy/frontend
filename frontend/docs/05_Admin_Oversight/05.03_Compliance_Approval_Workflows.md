# Compliance Approval Workflows
Version: v1.0 (LOCKED)  
Scope: Governance-only; no implementation

## Compliance Gates Before Payout
- Product certifications verified per category.
- Buyer/importer declarations collected where required.
- Sanctions/PEP checks cleared for supplier and buyer.
- Export/import licence checks for controlled goods.

## Rejection and Remediation
- If compliance fails, order moves to PAYMENT_REVIEW_REQUIRED or payout hold.
- Remediation actions documented: provide missing certs, update HS codes, correct invoices.

## Audit Traceability
- Every approval/rejection must record: approver, timestamp, rationale, linked documents.
- Changes to compliance status must be immutable and auditable.
